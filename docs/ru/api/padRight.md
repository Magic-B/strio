# padRight

Добавляет отступ справа до заданной длины.



## padRight

### Сигнатура

```typescript
function padRight(str: string, length: number, char?: string): string
```

### Примеры

```typescript
import { padRight } from 'strio'

padRight('5', 3, '0')
// "500"
```
