# pascalize

Преобразует строку в PascalCase.



## pascalize

### Сигнатура

```typescript
function pascalize(str: string): string
```

### Примеры

```typescript
import { pascalize } from 'strio'

pascalize('hello-world')
// "HelloWorld"

pascalize('hello_world')
// "HelloWorld"
```
