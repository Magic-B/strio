# randomAlpha

Генерирует случайную строку из букв.



## randomAlpha

### Сигнатура

```typescript
function randomAlpha(length: number): string
```

### Примеры

```typescript
import { randomAlpha } from 'strio'

randomAlpha(10)
// "aBcDeFgHiJ"
```
