# camelize

Преобразует строку в camelCase.



## camelize

### Сигнатура

```typescript
function camelize(str: string): string
```

### Примеры

```typescript
import { camelize } from 'strio'

camelize('hello-world')
// "helloWorld"

camelize('hello_world')
// "helloWorld"
```
