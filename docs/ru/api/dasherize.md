# dasherize

Преобразует строку в kebab-case.



## dasherize

### Сигнатура

```typescript
function dasherize(str: string): string
```

### Примеры

```typescript
import { dasherize } from 'strio'

dasherize('helloWorld')
// "hello-world"

dasherize('HelloWorld')
// "hello-world"
```
