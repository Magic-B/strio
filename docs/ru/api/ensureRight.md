# ensureRight

Гарантирует, что строка заканчивается заданным суффиксом.



## ensureRight

### Сигнатура

```typescript
function ensureRight(str: string, suffix: string): string
```

### Примеры

```typescript
import { ensureRight } from 'strio'

ensureRight('hello', '!')
// "hello!"
```
