# randomAlphanumeric

Генерирует случайную буквенно-цифровую строку.



## randomAlphanumeric

### Сигнатура

```typescript
function randomAlphanumeric(length: number): string
```

### Примеры

```typescript
import { randomAlphanumeric } from 'strio'

randomAlphanumeric(10)
// "a1B2c3D4e5"
```
