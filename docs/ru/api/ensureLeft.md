# ensureLeft

Гарантирует, что строка начинается с заданного префикса.



## ensureLeft

### Сигнатура

```typescript
function ensureLeft(str: string, prefix: string): string
```

### Примеры

```typescript
import { ensureLeft } from 'strio'

ensureLeft('world', 'hello ')
// "hello world"
```
