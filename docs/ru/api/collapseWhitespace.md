Схлопывает множественные пробелы в одинарные.



## collapseWhitespace

### Сигнатура

```typescript
function collapseWhitespace(str: string): string
```

### Примеры

```typescript
import { collapseWhitespace } from 'strio'

collapseWhitespace('hello    world')
// "hello world"
```
