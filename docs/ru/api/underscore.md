# underscore

Преобразует строку в snake_case.



## underscore

### Сигнатура

```typescript
function underscore(str: string): string
```

### Примеры

```typescript
import { underscore } from 'strio'

underscore('helloWorld')
// "hello_world"

underscore('HelloWorld')
// "hello_world"
```
