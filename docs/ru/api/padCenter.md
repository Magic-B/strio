# padCenter

Центрирует строку, добавляя отступы с обеих сторон.



## padCenter

### Сигнатура

```typescript
function padCenter(str: string, length: number, char?: string): string
```

### Примеры

```typescript
import { padCenter } from 'strio'

padCenter('hello', 11)
// "   hello   "
```
