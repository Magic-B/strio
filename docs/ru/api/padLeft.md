# padLeft

Добавляет отступ слева до заданной длины.



## padLeft

### Сигнатура

```typescript
function padLeft(str: string, length: number, char?: string): string
```

### Примеры

```typescript
import { padLeft } from 'strio'

padLeft('5', 3, '0')
// "005"
```
